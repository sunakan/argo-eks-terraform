== Terraform + EKS + ArgoWorkflow

=== EKS

.準備
----
$ cd terraform
$ terraform init
$ teerraform apply
10分くらいまつ
$ aws eks --region ap-northeast-1 update-kubeconfig --name eks-example
----

.確認
----
$ kubectl get all --all-namespaces
----
